caja-kissy
==========

kissy for caja, a third parth adapter.

caja 是google开发的安全技术框架 https://code.google.com/p/google-caja/
开放JS 是基于caja的修改版，更适合公司的应用。 https://github.com/lorrylockie/caja
caja-kissy 是基于开放JS的一个适配器，让第三方可以在开放JS中安全的使用KISSY相关特性。

caja-kissy主要关于与kissy对caja的适配，是kissy的一个安全策略方案。

目前caja-kissy和开放JS应用于淘宝的旺铺模板开发，淘宝U站，品牌中心等
